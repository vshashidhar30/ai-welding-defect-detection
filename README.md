# AI Welding Defect Detection System

An AI-powered welding defect detection system developed using YOLOv8, OpenCV, Python, and Tkinter GUI for real-time industrial inspection.

## Features

- Detects welding defects like:
  - Porosity
  - Spatter
  - Underfill

- Real-time defect detection
- GUI-based image upload system
- Displays causes and solutions for detected defects
- Saves results automatically
- Lightweight and suitable for low-spec systems

## Technologies Used

- Python
- YOLOv8
- OpenCV
- Tkinter
- Pillow
- Ultralytics

## Project Workflow

1. Image Collection
2. Annotation using Roboflow
3. Data Augmentation
4. YOLOv8 Model Training
5. GUI Development
6. Real-time Defect Detection

## Results

- mAP@50: 40%
- F1 Score: 30%
- Real-time detection performance
